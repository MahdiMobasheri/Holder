## ✨ Overview

Holder is a lightweight Chrome extension designed to make managing browser content easier.

It focuses on simplicity, speed, and a clean user experience while running directly inside your browser.

No complicated setup.
No external services required.
Just a simple browser tool.

---

## 🚀 Features

- ⚡ Lightweight and fast
- 🧩 Chrome Extension based
- 🔒 Runs locally in your browser
- 📦 No account required
- 🌐 No backend dependency
- 🎯 Simple and clean interface
- 🛠️ Easy to customize

## 📥 Installation

### Using Release (Recommended)

1. Open the **Releases** section
2. Download the latest ZIP file
3. Extract the downloaded file
4. Open Chrome and go to:

```

chrome://extensions/

```

5. Enable **Developer mode**
6. Click **Load unpacked**
7. Select the extracted folder

The extension is ready to use 🎉

---

### Using Download ZIP

1. Click:

```

Code → Download ZIP

```

2. Extract the file
3. Load the extracted folder from Chrome Extensions page
